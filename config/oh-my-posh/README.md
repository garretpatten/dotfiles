# Oh My Posh

This directory is kept so `shell.sh` can sync `config/oh-my-posh/` to
`~/.config/oh-my-posh` on fresh installs.

The prompt theme is **`gruvbox.omp.json`**, bundled with Oh My Posh (see
`home/zsh/darwin.zsh`). Add custom `.omp.json` files here if you want to
override the stock theme path.
